Сопартицы:
	Воин:
		Побеждает одного скелета, лтбо одного слизня, либо любое количество слизней
		Открывает только один сундук
		Пьет любое количетво зелий
	Клирик:
		Побеждает одного гоблина, лтбо одного слизня, либо любое количество скелетов
		Открывает только один сундук
		Пьет любое количетво зелий
	Маг:
		Побеждает одного гоблина, либо одного скелета, либо любое количество слизней.
		Открывает только один сундук
		Пьет любое количетво зелий
	Вор:
		Побеждает одного гоблина, либо одного скелета, либо одного слизня.
		Открывает любое количество сундуков
		Пьет любое количетво зелий
	Страж:
		Побеждает любое количество монстров одного типа за бой (скелетов, гоблинов или слизней).
		Открывает любое количество сундуков
		Пьет любое количетво зелий
	Свиток:
		Перебросить любое количество кубиков подземелья и партии
		Пьет любое количетво зелий
Монстры:
	Гоблин
	Скелет
	Слизень
	Сундук
		Позволяет взять 1 сокровище
	Зелье
		Позволяет вернуть 1 кубик партии с кладбища (воскресить 1 сопартийца)
	Дракон
		Отправляется в логова дракона. Если в логове дракона 3 и более кубиков - начинается битва с драконом
		Для победы над драконом нужно 3 разных сопартийца
